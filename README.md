# ServerTP - Proxy сервер для Яндекс.Диска

Этот сервер позволяет скачивать файлы из публичной папки Яндекс.Диска.

## Использование

1. Сервер запускается на порту 3000 (или порту, указанном в переменной окружения PORT)
2. Для скачивания файла используйте: `/download/имя_файла`
3. Пример: `https://ваш-сервер.railway.app/download/report.xlsx`

## Тестирование локально

```bash
npm install
npm start
